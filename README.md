## 📖 Overview

Welcome to my personal portfolio repository! This project serves as a centralized hub to demonstrate my engineering capabilities, academic research, and technical achievements. 

Designed with an emphasis on premium aesthetics and seamless interactivity, this repository showcases:
- **AI Research**: Academic publications and multimodal deep learning explorations.
- **Featured Projects**: Complete full-stack implementations of federated learning and generative AI architectures.
- **Skills & Certifications**: Validated credentials from world-class institutions like IIT Madras, Microsoft, and Infosys.
- **Experience**: Timelines of my professional internships, mentoring roles, and technical hackathons.

## ✨ Features

- **Premium Glassmorphism UI**: A sleek, translucent aesthetic built natively with Tailwind CSS.
- **Interactive 3D Hero**: Features an immersive, interactive particle sphere powered by `@react-three/fiber`.
- **Dark & Light Mode**: Flawless dynamic theming utilizing `next-themes`.
- **Responsive Design**: Mobile-first architecture that scales gracefully across all viewports.
- **Publications & Research**: A dedicated section to cross-link academic abstracts with full-stack code implementations.
- **Featured Research & Work**: Expandable, highly-detailed case studies for flagship engineering projects.
- **GitHub Dashboard**: Live, visual extraction of GitHub commits and language statistics.
- **Certifications & Achievements**: A fully data-driven, minimalistic credential showcase.
- **SEO Optimized**: Built with Next.js App Router for superior server-side rendering and meta-tagging.
- **Accessibility Focused**: Engineered with semantic HTML and comprehensive ARIA labels.

## 🛠️ Technology Stack

### Frontend & UI
- **Framework**: Next.js (App Router)
- **Library**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Rendering**: Three.js & React Three Fiber
- **Icons**: Lucide React & React Icons

### Backend & Tooling
- **API Routes**: Next.js Serverless Functions
- **Bundler**: Turbopack
- **Package Manager**: npm

## 📂 Project Structure

```text
src/
├── app/                  # Next.js App Router entry points & global styles
│   ├── api/              # Serverless API routes (Contact form handler)
│   ├── globals.css       # Global Tailwind directives
│   ├── layout.tsx        # Root layout & provider wrappers
│   └── page.tsx          # Main landing page assembling all sections
├── components/           # Reusable UI architecture
│   ├── canvas/           # Three.js 3D components (NeuralSphere)
│   ├── providers/        # Context providers (Theme, Smooth Scroll)
│   ├── sections/         # Individual portfolio sections (Hero, Projects, Research, etc.)
│   └── ui/               # Core atomic components (Navigation, Footer, MagneticButton)
├── data/                 # The single source of truth! (All text, URLs, and arrays)
│   └── index.ts          # Centralized data file for easy content management
├── lib/                  # Utility functions
│   └── utils.ts          # Tailwind merge utilities (clsx, twMerge)
└── types/                # Global TypeScript declarations
```

## 🚀 Getting Started

To run this portfolio locally on your machine, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/PreranRai/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

4. **Build for production**
   ```bash
   npm run build
   ```
